## AI-Powered Smart Waste Recommendation System

This project uses a CNN-based image classifier to identify waste types and a
rule-based sustainability engine to recommend eco-friendly disposal methods.

### Current Features
- Waste classification (Plastic, Paper, Metal)
- Sustainable disposal recommendation
- Eco-score generation
- Environmental tips for users

### Tech Stack
- Python
- TensorFlow/Keras (CNN)
- Rule-based decision engine
